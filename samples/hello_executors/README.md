# Executors
Executors API available for C++, C, C#, Python

Each project has their own README.md for details. All this executors just returns ```Hello from \<language name\>``` as JSON on GET request at http://127.0.0.1:8080

## C++
Endpoints:
* http://127.0.0.1:8080/
* http://127.0.0.1:8080/cxx

Additonal information: [README.md](cxx/README.md)

## C
Endpoint:

Additonal information: [README.md](cc/README.md)
